# 🍽️ Python Cafe

A simple and interactive restaurant web application built using **Python (Flask)**, where users can browse menu items, add them to cart, and place an order.

---

## 🚀 Features

* 🧾 Dynamic menu display (Pizza, Pasta, Burger, Salad, etc.)
* ➕ Add items with quantity (+ / − controls)
* 🛒 Multiple item selection (cart-like system)
* 💰 Calculates total bill
* 🎨 Attractive UI with responsive design
* ⚡ Real-time interaction using JavaScript

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Templating Engine:** Jinja2

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/yashbora18/pythoncafe-repo.git
```

2. Navigate to project folder:

```
cd python-cafe
```

3. Install Flask:

```
pip install flask
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000
```
